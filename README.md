## Tech Stack

* `NodeJS` `16+`
* `Golang` `17+`
* The `bash` `B`ourne `A`gain `SH`ell, version above `GNU bash, version 4.4.12`
* The [`hugo`](https://gohugo.io/) extended cli,  version above `hugo v0.89.4-AB01BA6E+extended` (`AB01BA6E` is a git commit hash `--short`)
